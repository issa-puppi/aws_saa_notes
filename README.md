# AWS SAA Study Notes Repository Overview

This repository contains structured notes and hands-on lab artifacts created while preparing for the AWS Certified Solutions Architect – Associate (SAA) certification.

It serves as both a personal knowledge base and a demonstration of practical AWS experience through applied examples and code.

---

## Structure

Each module is organized into its own directory, focused on a specific AWS topic.

Within each module:

- **`*.notes.md`**  
  Structured notes covering key concepts, explanations, and references to official documentation and external resources

- **`code/`**  
  Scripts, command examples, and lab-based implementations used to reinforce practical understanding

- The final module includes a `quick-reference-glossary.md` that consolidates key links and resources for each topic, serving as a convenient review tool

---

## Purpose

This repository is intended to:

- Reinforce understanding of AWS services through written and applied learning  
- Track progress throughout the SAA certification journey  
- Provide a reusable reference for future projects and implementations  
- Demonstrate hands-on experience with AWS concepts and tooling  

---

## Topics Covered

- Identity and Access Management (IAM)  
- EC2 and Compute Services  
- Networking (VPC, Subnets, Security Groups)  
- Storage (S3, EBS, EFS)  
- Monitoring and Logging  
- Additional core SAA domains
