# AWS SAA Study Notes Repository

This repository contains notes and hands-on lab artifacts created while studying for the AWS Certified Solutions Architect – Associate (SAA) certification.

It serves both as a personal knowledge base and as a practical demonstration of AWS concepts through real examples and code snippets.

## Structure

Each module is organized into its own directory, focused on a specific AWS topic.

Within each module directory:

* **`notes.md`**
  Contains structured notes covering key concepts, explanations, and references to official documentation and external learning resources.

* **`code/`**
  Contains scripts, command examples, and code snippets developed during hands-on labs and exercises. These are intended for review, reuse, and practical reinforcement of concepts.

## Purpose

This repository is designed to:

* Reinforce understanding of AWS services through written notes and applied examples
* Track learning progress throughout the SAA certification journey
* Provide a reusable reference for future projects and real-world implementations
* Demonstrate practical experience with AWS concepts and tooling

## Topics Covered

* Identity and Access Management (IAM)
* EC2 and Compute Services
* Networking (VPC, Subnets, Security Groups)
* Storage (S3, EBS, EFS)
* Monitoring and Logging
* And other core SAA domains

## Notes

* Code snippets are primarily intended for educational and reference purposes
* Some scripts are designed to run specifically in AWS environments (e.g., EC2 user data)
* Always review and adapt configurations before using in production environments
